# Папка для хранения загруженных файлов

Эта папка используется для хранения документов, загруженных через Telegram бота.

## Структура

```
uploads/
└── documents/     # Карточки компаний и другие документы
```

## Именование файлов

Файлы сохраняются с именем на основе SHA256 хеша содержимого:
- Формат: `{sha256_hash}{расширение}`
- Пример: `da9e69aaf24f437f3cdb97192649cb9134b275b434cd7c0cbb49e336432c1fe4.pdf`

Это позволяет:
- Автоматически избегать дубликатов
- Безопасно хранить файлы (имена не содержат специальных символов)
- Легко находить файлы по хешу из базы данных

## База данных

Информация о файлах хранится в таблице `company_document`:
- `file_name` - оригинальное имя файла
- `file_sha256` - хеш файла
- `storage_url` - полный путь к файлу на диске

## Бэкап

⚠️ **Важно**: Регулярно делайте резервные копии этой папки!

Файлы НЕ восстанавливаются автоматически из базы данных.
